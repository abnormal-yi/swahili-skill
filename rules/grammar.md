## Sentence Structure & Grammar (Sarufi)

### Word Order: SVO

**Correct:**

```swahili
Mimi ninakula chakula.    // I am eating food.
Yeye anasoma kitabu.       // S/he is reading a book.
```

### Adjectives Follow Nouns

**Incorrect (reversed order):**

```swahili
zuri kitabu               // ❌ Wrong order
```

**Correct:**

```swahili
kitabu kizuri             // ✅ book good (adjective after noun)
mtoto mzuri               // ✅ child good
```

### Possessives

| Person | Singular | Plural |
|--------|----------|--------|
| 1st | -angu (my) | -etu (our) |
| 2nd | -ako (your) | -enu (your) |
| 3rd | -ake (his/her) | -ao (their) |

Must agree with noun class:
- kitabu **changu** (ki-class → ch-)
- nyumba **yangu** (n-class → y-)
- kitu **changu** (ki-class → ch-)

### Demonstratives

| Distance | Ki- class | Vi- class |
|----------|-----------|-----------|
| Near | hiki kitabu | hivi vitabu |
| Mid | kile kitabu | vile vitabu |
| Far | hicho kitabu | hivyo vitabu |
