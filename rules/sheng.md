## Sheng (Mchanganyiko wa Swahili na Kiingereza)

Sheng is a Swahili-English urban slang originating in Nairobi, Kenya, now used across East Africa.

### Common Sheng Terms

| Sheng | Meaning | Notes |
|-------|---------|-------|
| niaje? / mamboo? | What's up? | Greeting |
| poa / fresh | Cool / good | Response |
| mboch / boch | Nothing / no | "Bosh" |
| msee | Guy / person | Also "jamless" |
| manzi | Girl | Female |
| kamata / chop | Get / understand | "Kamata" = catch |
| finje | Fake / poor quality | |
| sherehe | Party / fun | From Swahili |
| mbaba | Father / old man | |
| buibui | Money | Literally "spider" |

### Usage Note

Sheng is **informal** — use standard Swahili for professional, formal, or academic contexts. Sheng is appropriate in casual conversation, pop culture, and with younger speakers.

**Incorrect (Sheng in formal context):**

```swahili
Msee huyo ananikamata.   // ❌ Too casual for formal
```

**Correct (standard in formal context):**

```swahili
Mtu huyo ananielewa.     // ✅ Standard Swahili
```
