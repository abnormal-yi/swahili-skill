## Noun Classes (Ngeli za Swahili)

Swahili has **16 noun classes** indicated by prefixes. Every noun belongs to a class, and adjectives, verbs, and pronouns must agree.

### Class Table

| Ngeli | Prefix | Mfano (Example) | Agreement Marker |
|-------|--------|----------------|------------------|
| 1 (M-) | m-/mw- | mtu (person) | a-, yu- |
| 2 (WA-) | wa-/w- | watu (people) | wa- |
| 3 (M-) | m-/mw- | mti (tree) | u- |
| 4 (MI-) | mi-/my- | miti (trees) | i- |
| 5 (JI-) | ji-/j- | jicho (eye) | li- |
| 6 (MA-) | ma-/m- | macho (eyes) | ya- |
| 7 (KI-) | ki-/ch- | kitu (thing) | ki- |
| 8 (VI-) | vi-/vy- | vitu (things) | vi- |
| 9 (N-) | n-/ny-/m- | nyumba (house) | i- |
| 10 (N-) | n-/ny-/m- | nyumba (houses) | zi- |
| 11 (U-) | u-/w- | uso (face) | u- |
| 14 (U-) | u-/w- | uzuri (beauty) | u- |
| 15 (KU-) | ku-/kw- | kula (to eat) | ku- |
| 16 (PA-) | - | pale (there) | pa- |
| 17 (KU-) | - | kule (over there) | ku- |
| 18 (MU-) | - | mulimo (inside) | mu- |

### Agreement Pattern

**Incorrect (class mismatch):**

```swahili
kitu mzuri       // ❌ kitu (class 7) + mzuri (class 1/3)
```

**Correct (class agreement):**

```swahili
kitu kizuri       // ✅ kitu (class 7) + kizuri (class 7)
vitu vizuri       // ✅ vitu (class 8) + vizuri (class 8)
mtu mzuri         // ✅ mtu (class 1) + mzuri (class 1)
watu wazuri       // ✅ watu (class 2) + wazuri (class 2)
```
