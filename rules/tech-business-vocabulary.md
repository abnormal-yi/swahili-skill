# Tech, Business & Tourism Vocabulary

East African professionals (especially freelancers, devs, and tourism operators) heavily code-switch: most technical nouns stay in English, while verbs, connectors, and everyday speech remain Swahili/Sheng. This file gives the realistic register used in Tanzania (and broadly Kenya/Uganda) for tech and business work.

## Common Code-Switched Sentences (Yichang-style shorthand)

These reflect realistic terse, mixed-register messages common among TZ freelancers:

| Sentence | Meaning |
|---|---|
| Nimefix bug ile ya filter | I fixed that filter bug |
| Plugin imekuwa active sasa, lakini design bado haijawa sawa na reference | The plugin is now active, but the design doesn't match the reference yet |
| Tuma screenshot ya error | Send a screenshot of the error |
| Niko na deadline kesho, tuongeze speed | I have a deadline tomorrow, let's speed up |
| Hii section ya accommodation bado ina tatizo la ACF fields | This accommodation section still has an ACF fields issue |
| Naomba uangalie kwenye mobile, design imevunjika | Please check on mobile, the design is broken |
| Tutaweka shortcode hii kwenye page ya Serengeti | We'll put this shortcode on the Serengeti page |

## Regional Notes (TZ vs KE)

- "Poa" is common in both Tanzania and Kenya for "cool/fine."
- "Fiti" is more Kenyan Sheng for "fine/great"; less common in Tanzania.
- Tanzanian Swahili leans more formal/standard (closer to Standard Swahili from Zanzibar); Kenyan Sheng mixes more English and tribal-language influence (Sheng proper is largely Nairobi-based).
- "Bongo" (slang for Tanzania, especially Dar es Salaam) is widely used; "Bongo Fleva" = Tanzanian hip-hop/pop culture, often a source of slang.
