## Common Everyday Tech Sentences

| Sentence | Meaning |
|---|---|
| Simu yangu haina mtandao | My phone has no network/signal |
| Nimeisha bando, naomba unitumie vocha | I've run out of data, please send me airtime/credit |
| Tuma kwa WhatsApp tu | Just send it on WhatsApp |
| Battery inaisha, nita-charge kidogo | The battery is dying, I'll charge it a bit |
| Bonyeza hapa uone | Click here to see |
| Sijapokea ujumbe wako | I haven't received your message |
| Nitumie screenshot ya hiyo error | Send me a screenshot of that error |
| Niko offline kwa sasa | I'm offline right now |
| Site inachukua muda kufungua | The site is taking a long time to load |
