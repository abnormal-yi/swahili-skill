## Core Vocabulary (Msamiati)

### Greetings (Salamu)

| Swahili | English |
|---------|---------|
| Habari / Hujambo | Hello |
| Shikamoo | Respectful greeting to elder |
| Marahaba | Response to Shikamoo |
| Mambo? (Sheng) | What's up? |
| Kwaheri | Goodbye |
| Tutaonana | See you later |
| Lala salama | Good night |
